# Ragdoll.js
[Live Demo](https://prashanthkumar0.github.io/ragdoll.js/)

<br/> 

This is just a Simple vanilla HTML5 canvas application.
Inspired by [THE LAST EXPERIENCE](https://codepen.io/ge1doot/pen/LZdOwj) pen.
It simply Uses Verlet integration with hit and trial but carefully choosen weighted time varying force to keep the character up on toes and make it dance.
